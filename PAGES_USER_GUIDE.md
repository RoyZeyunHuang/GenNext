# Ops Hub 页面功能与用户流程说明

本文档面向产品、运营或新同事，简要说明每个页面的功能以及用户典型操作流程。不涉及技术实现细节。

---

## 概览（Dashboard）

**路径**：`/dashboard`  
**一句话**：工作台首页，一眼看到本周要关注的事。

**功能**：
- 顶部根据当前时间显示问候语（如「上午好」）。
- **日历卡片**：展示近期/今日日历事件入口或摘要。
- **新闻卡片**：今日行业简报与社媒选题的快捷入口。
- **待办卡片**：待办列表，可勾选完成。
- **KPI 卡片**：本周 KPI 进度条或关键指标。

**用户流程**：
1. 进入应用后从侧栏点「概览」进入。
2. 扫一眼各卡片，了解今日/本周待办、日程和 KPI。
3. 需要时点击对应卡片或侧栏进入「日历」「新闻」「KPI」等详情页。

## 内容工厂（Documents）

**路径**：`/documents`  
**一句话**：统一管理品牌档案、知识库、任务模板和人格模板，供「内容创作」页使用。

**功能**：
- **品牌档案**：按楼盘或全局维护品牌介绍、卖点、话术等；支持上传 TXT/DOCX/PDF，自动解析为文本内容。
- **知识库**：存放产品知识、行业资料、FAQ 等，可打标签、填来源链接。
- **任务模板**：预设「小红书种草」「Instagram Caption」「LinkedIn 专业帖」「微信跟进」等写作要求与格式。
- **人格模板**：预设人设（如专业人设、生活博主人设），控制文案语气与风格。

**用户流程**：
1. 在顶部 Tab 切换「品牌档案 / 知识库 / 任务模板 / 人格模板」。
2. **新增**：点「新建」，填标题、内容（或上传文件），必要时选楼盘、平台、是否默认等，保存。
3. **编辑/删除**：在列表中点某条进入编辑或删除。
4. 在「内容创作」页写文案时，会从这里勾选要用的档案和模板；平时只需维护好这些资料即可。

## 内容创作（Copywriter）

**路径**：`/copywriter`  
**一句话**：输入需求，选好档案与模板，由 AI 生成营销文案并可保存、收藏。

**功能**：
- 输入自然语言需求（如「给 XX 楼盘写一篇小红书种草」）。
- **智能推荐**：点「智能推荐」后，AI 根据输入自动推荐合适的品牌档案、知识库、任务模板、人格模板，并自动勾选。
- 可手动增删「品牌档案」「知识库」「任务模板」「人格模板」。
- **生成**：点「生成」后流式输出文案，可复制、保存到历史、标星收藏。
- **历史**：查看最近生成的文案列表，可再次复制或查看当时用的配置。

**用户流程**：
1. 在输入框写下需求（楼盘、平台、风格等）。
2. （可选）点「智能推荐」，确认或微调自动勾选的档案与模板。
3. 点「生成」，等待文案逐字输出。
4. 满意则「复制」或「保存到历史」；重要的一条可「标星」便于后续查找。
5. 需要时在下方「历史」中查看或复用以往生成结果。

## AI 日历（Calendar）

**路径**：`/calendar`  
**一句话**：用文字或图片描述日程，AI 解析成日历事件并写入日历、可下载 .ics。

**功能**：
- **文字解析**：在文本框输入一段话（如「下周三下午 3 点团队会议，会议室 A」），AI 提取时间、标题、地点等。
- **图片解析**：上传一张包含日程信息的截图或海报（JPG/PNG），AI 识别后输出结构化事件。
- 解析结果以列表展示，可**批量写入**系统日历，并**下载 .ics** 文件导入其他日历应用。
- 页面内可查看**已保存的日历事件列表**，支持删除单条。

**用户流程**：
1. 选择「文字」或「图片」输入方式。
2. 输入文字或拖拽/选择一张图片，点「解析」或「生成」。
3. 查看解析出的事件列表，确认或微调（若有编辑能力）。
4. 点「保存到日历」写入系统；需要时点「下载 .ics」到本地。
5. 在「历史事件」中查看、管理已保存事件，不需要的可删除。

## BD CRM

**路径**：`/crm`  
**一句话**：管理楼盘、公司、联系人及跟进状态，带简单看板与 AI 跟进话术。

**功能**：
- **楼盘**：维护在跟进的楼盘信息（名称、地址、面积、价格区间等）。
- **公司**：维护合作方/客户公司，关联联系人。
- **联系人**：每个公司下可有多位联系人，含职位、电话、邮箱、LinkedIn 等。
- **跟进（管线）**：每条跟进关联一个楼盘，有状态（如 Not Started / 沟通中 / Won / Lost）、联系人、备注；看板或列表展示。
- **看板统计**：总楼盘数、跟进数、赢单/输单、赢率、平均成交天数、各状态数量、近 12 周趋势等。
- **AI 跟进话术**：在跟进详情或列表中，根据公司名和近期备注，一键生成适合微信发送的跟进话术。

**用户流程**：
1. 在「楼盘」「公司」「联系人」Tab 中先维护好基础数据。
2. 在「跟进」Tab 里新建跟进，选择楼盘、填联系人与备注，并更新状态。
3. 通过看板或统计了解当前管线健康度与趋势。
4. 需要发微信跟进时，在对应跟进条上使用「AI 生成话术」，复制后到微信发送。

## KPI

**路径**：`/kpi`  
**一句话**：上传小红书/Instagram 等数据，查看自然流、付费、报表与排名，支持楼栋/AE 维度和 AI 分析。

**功能**：
- **数据上传**：  
  - **自然流（小红书）**：上传含标题、发布时间、曝光、播放、点赞等字段的数据；系统按标题+时间或标题匹配帖子，自动新建或关联帖子并写入当日快照。  
  - **付费**：按笔记 ID 上传消耗、曝光、点击等付费指标，按日汇总。  
  - **Instagram**：上传 IG 帖子与指标，写入 IG 帖子表与快照。
- **概览**：按时间、楼栋、AE 等筛选，查看整体或单帖的自然流/付费/IG 表现。
- **报表**：创建或查看「活动报表」，包含时间范围、汇总数据、Top 帖子等。
- **楼栋/AE 列表**：基于帖子属性（楼栋、AE）做筛选与统计。
- **AI 分析**（若有）：对选定数据做简要解读或建议。

**用户流程**：
1. 在「数据上传」Tab 选择类型（自然流/付费/IG），按模板整理 Excel 或 JSON，上传。
2. 系统展示本次导入条数、新建帖子数等；若有重复或歧义会提示。
3. 在「概览」「Top 帖子」「报表」等 Tab 中按日期、楼栋、AE 筛选查看数据。
4. 需要汇报时在「活动报表」中生成或查看某次活动的汇总与 Top 内容。
5. 需要维度时使用「楼栋列表」「AE 列表」做交叉分析。

## 新闻摘要（News）

**路径**：`/news`  
**一句话**：看每日行业简报与社媒选题，把感兴趣的保存为「文案灵感」。

**功能**：
- 页面拉取**当日**行业简报与社媒选题（来自配置的每日新闻 API）。
- 分块展示「行业简报」「社媒选题」，每条有标题、来源、摘要或钩子。
- 每条可点「转为文案灵感」：将当前条保存到系统「新闻/灵感」库（带来源、摘要、标签），便于后续在写文案时参考。

**用户流程**：
1. 从侧栏进入「新闻摘要」，等待或点击「刷新新闻」加载当日内容。
2. 浏览行业简报与社媒选题。
3. 看到适合做选题或参考的，点「转为文案灵感」保存。
4. 保存过的条目标记为「已保存」，避免重复保存。

## 全局 AI 助手（右下角浮窗）

**位置**：全站右下角，任意页面可用  
**一句话**：随时可用的通用 AI 对话，不绑定具体业务数据。

**功能**：
- 点击浮窗展开输入框，输入问题或指令。
- 支持多轮对话，AI 以流式方式回复。
- 与「内容创作」区别：这里不自动带入品牌档案、知识库或模板，适合问通用问题、头脑风暴、翻译、写简单文案等。

**用户流程**：
1. 在任何页面点击右下角 AI 助手图标展开。
2. 输入问题或需求，发送。
3. 查看流式回复，可继续追问或关闭浮窗。
4. 需要「带品牌/模板的正式文案」时，应去「内容创作」页使用智能推荐与生成。

## 侧栏与语言切换

- **侧栏**：固定展示「概览、内容工厂、内容创作、AI 日历、BD CRM、KPI、新闻摘要」入口，点击即跳转对应页面。
- **语言切换**：侧栏底部提供「中文 / EN」切换，用于界面语言（若已实现多语言）。
