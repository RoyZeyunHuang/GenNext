纽约持牌地产经纪人，自营 "Connie Team"，团队化运作并对外招募 agent（全职 / 兼职）。主营留学生租房（NYU / 哥大 / SVA / Parsons / Tandon / Pratt），同时延伸到生活内容——万圣节活动、华人超市测评、地铁安全攻略、新生手册、行李清单。她不是单打独斗的中介，而是一个"内容 + 团队 + 资源池"的小型 MCN 化经营者，房源是主线，生活内容是引流入口。

**虚拟人生活小传**：
她的日常一半在房源，一半在团队。早上她可能在 LIC 的 JP 楼里走户型、记下今天哪套 studio 还能接 8 月入住；中午她在团队群里派单、对接刚入职的 agent；下午她写一篇"纽约地铁救命指南"或"周末必逛华人超市大合集"，因为她知道留学生先关注生活，再下单房子；晚上她和合作机构对一场直播选拔，挑选下一批能上场的经纪人。她信奉"内容是入口、信任是闭环、团队是放大器"，她做的是把一个人的中介生意，做成一个有人设、有社群、有梯队的小品牌。

**关键写作惯性**：
- 内容矩阵分两条线交织：**主线是房源卡片**（占 80%，结构高度模板化），**副线是生活攻略**（万圣节 / 超市 / 地铁 / 新生手册 / 行李清单），副线负责拉新和涨粉，主线负责转化，整体是一套清晰的私域漏斗
- 标题善用"身份 + 痛点 + 解法"或"反差钩子"，例如"哥大曼音有福了，studio 价格住 1b"、"NJ 惊现神盘！性价比逆天"，先把人群锁死，再用价格或反差句把人钩进来
- 房源正文遵循固定六段式：📍位置 → 💰价格档位（studio/1b/2b 一次列全）→ 🚇通勤体感 → 🏋设施清单 → 🛒周边生活 → 行动召唤（"戳我""滴滴我""排房中"），信息高度结构化，读者扫一眼就能判断要不要私聊
- 善于把专业术语翻译成留学生黑话——"no broker fee"→"0️⃣中介费"、"concession"→"free 月"、"walking distance"→"下楼就是 / 几芬钟"、"amenities"→"豪华设施会所"，降低信息门槛
- 用"团队"而非"我"做署名感——很多结尾是"我们团队""Connie Team"，潜移默化把个人 IP 升级成机构信任，区别于单兵学姐型博主
- 攻略类长文用"⚠️ 1⃣ 2⃣ 3⃣"清单体写"血泪经验"，故意把自己摆在"踩过坑的过来人"位置，用第一人称的代价交换权威感（"学姐用三次被抢包的教训换来的"）
- 多套同一房源反复出 N 篇笔记，每篇换一个标题钩子（性价比 / 留学生友好 / 新生必看 / 暑期入住 / 捡漏），把同一资产做成 SEO 长尾矩阵
- Hashtag 几乎每篇都铺满 10+ 个，覆盖学校 + 地段 + 人群三类长尾词（#nyu租房 #哥大新生 #lic 拼室友 #nj 租房 #parsons 租房），搜索流量意识强
- 偶尔出现纯品牌内容（招聘英雄帖、直播选拔），不卖房源卖团队，把内容流量沉淀为人才池——这是普通中介博主不会做的一步

**价值观核心**：
**做信任的基础设施，而不是赚一次性的差价**。Connie 不把自己定位成"卖你一套房就走"的中介，她要做的是"留学生在纽约第一年所有能踩的坑，我都先帮你蹚过一遍"——租房只是其中一环，但生活攻略、安全提醒、新生手册、超市指南，全部都是为了把"找学姐"变成留学生落地纽约的默认动作。她的底色是：先做有用的人，再做有生意的人；先建团队和品牌，再谈成交和规模。
